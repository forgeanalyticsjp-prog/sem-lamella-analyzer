# SEM Analyzer

> SEM画像から、ラメラ状組織の **太さ・長さ・間隔・空白域** を自動で定量化するデスクトップツールです。
> 測定者ごとのばらつきをなくし、全画像を同一手法で一括処理します。

---

## 解決する課題

| | |
|---|---|
| **課題** | SEM組織の評価は手作業に頼ると、測定者・測定回数ごとに結果がブレ、試料間の定量比較が困難。とくに「組織の不均一性」は従来手法では数値化しにくい。 |
| **解決** | スケールバーを自動検出してµm換算し、全画像に同一アルゴリズムを適用。ラメラの幾何指標に加え、**最寄りラメラまでの距離マップから空白域（ラメラ希薄領域）を定量化**することで、平均値では見えない組織の不均一性を捉える。 |
| **成果** | フォルダを指定するだけで全画像を一括処理。指標をCSVとパネル画像で出力し、属人性を排した再現性のある比較が可能になる。 |

---

## 測定指標

| 指標 | 内容 | 手法 |
|---|---|---|
| ラメラ太さ（中央値） | ラメラ1本の太さ | 骨格上の距離変換 × 2 |
| ラメラ長さ（中央値 / 95%点） | ラメラ長さの分布 | 連結成分の主軸長 |
| ラメラ間隔 | ラメラ同士の間隔 | 線分法（水平・垂直のテスト線で横切り数を計数） |
| 空白域率 | ラメラが希薄な領域の面積割合 | 最寄りラメラまでの距離がしきい値（既定1.5µm）を超える領域 |
| 最大空白域パッチ | 最も大きい空白域の面積 | 空白域の連結成分の最大値 |
| 距離 95% / 99%点 | 組織の不均一性の指標 | 距離マップのパーセンタイル |

> **空白域指標がこのツールの差別化ポイントです。**
> ラメラの平均間隔や長さが同じでも、組織の「ムラ」の大きさは試料ごとに異なります。
> この不均一性を距離マップから定量化することで、材料特性との相関分析が可能になります。

## 出力例

### サンプル — 暗ラメラ（Dark lamellae）、1,000×

SEM画像でラメラが暗く写る典型的なケース。スケールバー・倍率はバナーから自動検出します。

| 入力 | 出力パネル |
|---|---|
| ![input_A](examples/sample_A/input.jpg) | ![panel_A](examples/sample_A/panel_all_metrics.png) |

```
ラメラ太さ（中央値）  : 0.299 µm
ラメラ長さ（中央値）  : 1.65 µm
ラメラ間隔           : 2.805 µm
空白域率             : 0.19 %
最大空白域パッチ      : 3.0 µm²
距離 95%点           : 0.911 µm
```

> **コントラスト反転（明ラメラ）にも自動対応しています。**
> 暗・明の向きはOtsu法で自動判定するため、設定変更は不要です。

## 機能

- **スケールバー自動検出** — 画像下部のインフォバナーからスケールバーを読み取り、ピクセルをµmに換算
- **倍率のファイル名自動判定** — `sample_x1000.jpg` のようなファイル名から倍率を推定
- **コントラスト反転対応** — 暗ラメラ・明ラメラをOtsu法で自動判定
- **バッチ処理** — フォルダを指定するだけで全画像を一括処理
- **GUI / CLI 両対応** — Pythonが入っていない環境向けのexe版も作成可能
- **非ASCIIパス対応** — 日本語フォルダ名・ファイル名でも動作

## 制限事項

- ラメラ検出はOtsu法（低コントラスト時はCLAHE併用）に基づくため、極端に不鮮明な画像では検出精度が低下します
- α/β相の同定は行いません。明暗コントラストを幾何学的に計測するツールです
- 太さの測定値は画像解像度に近い領域では分解能の影響を受けます（パネル上に注記されます）
